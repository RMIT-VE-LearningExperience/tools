How to Use This Tool

  1. Set prefix (optional) — choose Week, Module, or Topic with a number, or turn it off
  2. Add files — enter a display name and paste the Canvas file URL for each
  3. Generate — copy the accordion HTML into your Canvas page
💡
Getting the Canvas file URL: Upload your file to Canvas first. Go to Files in your course, right-click the file name, then select Copy link address.

Supported URL formats

/courses/{id}/files/{id}

/courses/{id}/files/{id}?wrap=1

/courses/{id}/files/folder/...?preview={id}
