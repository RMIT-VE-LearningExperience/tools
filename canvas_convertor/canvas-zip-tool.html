Why this tool?
Windows often nests files inside a folder when zipping — Canvas needs the XML at the root level. This tool creates a guaranteed flat ZIP that works every time, on any OS.

How it works

Select your downloaded text or XML file → confirm the filename (auto-renamed to .xml) → a flat ZIP downloads instantly. Upload it straight to Canvas. No software to install, works on Windows, Mac & Linux.
