How to Use This Tool

  1. Enter your program details (qual code, program code, cluster)
  2. Enter your course details (name, codes, welcome text)
  3. Paste the homepage HTML from the build course
  4. Generate and copy the updated HTML back into Canvas
🔨
Program Mode
Updating multiple courses in the same program? Toggle on — qual code, program code, and cluster lock in place. Just update the course-specific fields, paste the new template, and generate.
💡
Getting the template HTML:

In Canvas, open the build course homepage, switch to HTML Editor view, select all (Ctrl+A) and copy (Ctrl+C).

What the tool replaces
