How to Use This Tool

  1. Upload the Assessor Guide (.docx) containing quiz questions
  2. Review the automatically extracted questions, answers, and correct responses
  3. Edit any questions, options, or correct answers as needed
  4. Generate QTI 1.2 XML ready for Canvas import
  5. Download the .xml file
  6. Zip the .xml file using the Flat Zipper tool
  7. Import into Canvas: Settings → Import Course Content → QTI .zip File
💡
12 Question Types Supported: Multiple Choice, Multiple Answer, True/False, Matching, Fill in the Blank, Fill in Multiple Blanks, Multiple Dropdowns, Numerical Answer, Calculated/Formula, Essay, File Upload, and Text Only.
