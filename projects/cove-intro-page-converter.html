How to Use This Tool

  1. Configure output mode, palette, corner radius, and icons (Flex only)
  2. Paste your Copilot-generated introduction page into the sidebar
  3. Enter the week number
  4. Review the automatically parsed sections and edit as needed
  5. Generate Canvas-ready HTML
ℹ️
Emble Mode preserves the Emble component structure for in-house Canvas pages.
Flex Mode outputs pure inline-styled HTML for export to SCORM or other platforms.
💡
Tip: Use the strict Copilot prompt template with delimiters (──── Topic Overview ────, etc.) for reliable parsing. Click "Load Sample" to test.
