How to Use This Tool

  1. Configure output mode, palette, corner radius, and icon (Flex only)
  2. Paste your Copilot-generated Class Presentation intro into the sidebar
  3. Enter the week number, topic name, file name, and Canvas file URL
  4. Review the parsed introduction and usage tips
  5. Generate Canvas-ready HTML in your chosen mode
ℹ️
Emble Mode preserves the Emble component structure for in-house Canvas pages.
Flex Mode outputs pure inline-styled HTML for export to SCORM or other platforms.
💡
Tip: Upload your PPTX to Canvas first, then grab the file URL from the Files area. The tool extracts the course ID and file ID automatically. Click "Load Generic" to skip Copilot and use ready-made content.
