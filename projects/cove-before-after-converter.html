How to Use This Tool

  1. Configure output mode, palette, corner radius, and icons (Flex only)
  2. Paste your Copilot-generated Before & After Class activities
  3. Enter the week number and topic name
  4. Review the automatically parsed activities and edit as needed
  5. Generate two Canvas-ready pages — one Before, one After
ℹ️
Emble Mode preserves the Emble component structure for in-house Canvas pages.
Flex Mode outputs pure inline-styled HTML for export to SCORM or other platforms.
💡
Tip: This tool generates TWO separate HTML outputs — one Before Class page and one After Class page. Copy each individually into Canvas. The parser handles both label-on-own-line and inline Copilot formats automatically.
